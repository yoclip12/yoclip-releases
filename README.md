# yoclip
<img width="800" alt="image" src="https://github.com/user-attachments/assets/1a1e882b-5005-4966-97f6-a627dcafe687" />


* 치지직 스트리머 요룰레히님의 신청곡 클립을 정리해, 원하는 클립을 플레이리스트로 만들어 볼 수 있는 서비스 입니다.

**단순히 플레이 시간만 연결해 주기 때문에, 원활한 시청을 위해서는 채널 구독이 되어있어야 합니다. (미구독시 광고나옴)**


## 사용방법


### 1.로그인 

**구글 아이디로 소셜 로그인을 진행합니다.**

**소셜 로그인시 필요한 권한만 수집하며, 개인정보가 걱정되신다면 빈 구글계정을 만들어 이용 가능합니다.**
